## Descripción

OutdoorApp es una aplicación web desarrollada como Trabajo Final de Grado de la carrera Ingeniería en Software.

El sistema permite a los usuarios:

* Consultar rutas de senderismo.
* Visualizar información detallada de cada recorrido.
* Obtener recomendaciones de actividades outdoor.
* Inscribirse en actividades grupales.
* Administrar su perfil de usuario.

---

## Tecnologías utilizadas

* React
* Vite
* React Router DOM
* JavaScript
* CSS
* Bootstrap

---

## Requisitos para la ejecución

Para ejecutar correctamente el sistema es necesario contar con:

* Node.js
* npm

Instalar las dependencias mediante:

```bash
npm install
```

Luego iniciar el proyecto con:

```bash
npm run dev
```
