# Descripción

OutdoorApp es una aplicación web desarrollada como Trabajo Final de Grado de la carrera Ingeniería en Software.

El sistema tiene como objetivo facilitar la recomendación y organización de actividades outdoor, permitiendo a los usuarios consultar rutas, visualizar información detallada de recorridos, obtener recomendaciones, inscribirse en actividades grupales y administrar su perfil de usuario.

---

# Tecnologías de la Información y la Comunicación Aplicadas

Durante el desarrollo del sistema se emplearon las siguientes tecnologías:

* **Python:** Implementación de la lógica de negocio y funcionalidades del backend.
* **Flask:** Framework utilizado para el desarrollo del servidor web y la comunicación entre el frontend y la base de datos.
* **SQLite:** Base de datos relacional utilizada para almacenar la información del sistema.
* **React:** Biblioteca JavaScript utilizada para el desarrollo de la interfaz de usuario.
* **HTML5:** Lenguaje de marcado empleado para la estructura de las interfaces.
* **CSS3:** Utilizado para el diseño visual y la personalización de la interfaz.
* **Bootstrap:** Framework CSS utilizado para facilitar el diseño responsive y la construcción de componentes visuales.
* **JavaScript (ES6):** Lenguaje utilizado para implementar la lógica del lado del cliente.
* **Vite:** Herramienta utilizada para la creación y ejecución del proyecto React durante el desarrollo.

---

# Funcionalidades implementadas

* Inicio de sesión.
* Registro de usuarios.
* Validación de contraseñas.
* Consulta de rutas outdoor.
* Visualización del detalle de cada ruta.
* Recomendaciones de actividades.
* Inscripción a actividades grupales.
* Gestión del perfil del usuario.
* Edición de datos del perfil.
* Cierre de sesión.
